For any Depict product that's integrated, it is important to set up tracking so that our product performs correctly.

This guide will walk you through the steps to set up tracking for your Depict project using DPC (Depict Performance Client).

<Note>If you are also using `@depict-ai/react-ui` only do the step [Product Card Tracking](/api-guide/tracking/product-cards) and onwards.</Note>

<Info>For more detailed information see the [DPC Reference](/reference/performance-client/)</Info>
